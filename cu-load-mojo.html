cassandra:cu-load

Full name:

org.codehaus.mojo:cassandra-maven-plugin:3.6:cu-load

Description:

Loads a CassandraUnit DataSet into a Cassandra instance.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Since version: 1.2.1-2.
  • Binds by default to the lifecycle phase: pre-integration-test.

Required Parameters

Name Type Since Description
cassandraDir File 1.2.1-2 The directory to hold cassandra's database.
Default value is: ${project.build.directory}/cassandra.
classesDirectory File 1.2.1-2 The directory containing generated classes.
User property is: project.build.outputDirectory.
testClassesDirectory File 1.2.1-2 The directory containing generated test classes.
User property is: project.build.testOutputDirectory.

User property is: cassandra.jmxPort.
keyspace String 1.2.1-2 The keyspace against which individual operations will be executed
User property is: cassandra.keyspace.
listenAddress String 1.2.1-2 Address to bind to and tell other Cassandra nodes to connect to. You must change this if you want multiple nodes to be able to communicate! Leaving it blank leaves it up to InetAddress.getLocalHost(). This will always do the Right Thing if the node is properly configured (hostname, name resolution, etc), and the Right Thing is to use the address associated with the hostname (it might not be). Setting this to 0.0.0.0 is always wrong. Do not change this unless you really know what you are doing.
Default value is: 127.0.0.1.
logLevel String 3.5 Log level of cassandra process. Logging is performed via log4j2.
Default value is: ERROR.

Default value is: 512.

Default value is: 9042.

rpcAddress String 1.2.1-2 Address to use for the RPC interface. Do not change this unless you really know what you are doing.
Default value is: 127.0.0.1.
rpcPort int 1.2.1-2 Port to listen to for the RPC interface.
Default value is: 9160.
User property is: cassandra.rpcPort.
skip boolean 1.2.1-2 Skip the execution.
Default value is: false.
User property is: cassandra.skip.
startNativeTransport boolean 2.0.0-1 Enable or disable the native transport server. Currently, only the Thrift server is started by default because the native transport is considered beta.
Default value is: false.
User property is: cassandra.startNativeTransport.
stopKey String 1.2.1-2 Key to be provided when stopping cassandra
Default value is: cassandra-maven-plugin.
User property is: cassandra.stopKey.
stopPort int 1.2.1-2 Port to listen to for receiving the stop command over
Default value is: 8081.
User property is: cassandra.stopPort.
storagePort int 1.2.1-2 Port to listen to for the Storage interface.
Default value is: 7000.
User property is: cassandra.storagePort.
systemPropertyVariables Map 1.2.1-2 List of System properties to pass to the JUnit tests.
yaml String 1.2.1-2 In yaml format any overrides or additional configuration that you want to apply to the server.
